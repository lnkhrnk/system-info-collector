# Сбор информации о системе (System Info Collector)

## Описание
Приложение собирает базовую информацию о системе (CPU, RAM, OS), используя системные утилиты, и выводит ее в стандартный поток вывода (stdout) в едином JSON-формате.

## Технологии
*   **Java 17**
*   **Gradle** — система сборки
*   **Jackson** — библиотека для работы с JSON
*   **JUnit 5** — фреймворк для тестирования
*   **Checkstyle** — инструмент статического анализа кода

## Установка и настройка

### 1. Скачивание проекта
Склонируйте репозиторий на свой локальный компьютер с помощью Git:

Перейдите в папку проекта
```bash
git clone https://github.com/lnkhrnk/system-info-collector.git
cd system-info-collector
```
### 2. Конфигурация
Перед запуском приложения необходимо создать файл конфигурации app.properties в корневой папке проекта (в той же папке, где находятся файлы build.gradle и gradlew).

Создайте файл app.properties и вставьте в него следующее содержимое:
```bash
memory.output.unit=MB
include.network.info=false
```
### 3. Сборка и запуск

Для компиляции кода и создания исполняемого JAR-файла выполните команду в терминале из корня проекта:

```bash
./gradlew clean build
```

### 4. Запуск приложения

После успешной сборки JAR-файл появится в директории build/libs. Запустите его командой:

```bash
java -jar build/libs/system-info-collector-1.0.0.jar
```

При успешном запуске приложение выведет JSON-объект с информацией о вашей системе.
